A React form application that collects user information and displays it on the screen.

## Features
- Input fields:
  - Name
  - UID
  - Age
  - Phone Number
  - Email
- Submit button
- Reset button
- Controlled components using React state

## Technologies Used
- React
- JavaScript
- CSS
